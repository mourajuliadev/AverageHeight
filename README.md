# Average Height - C#

Projeto simples desenvolvido em C# para calcular a média de alturas informadas pelo usuário.

## Funcionalidades

- Leitura de múltiplas alturas
- Armazenamento em array
- Cálculo da média
- Exibição formatada do resultado

## Conceitos praticados

- Arrays
- Estruturas de repetição (`for`)
- Entrada de dados
- Conversão de tipos
- Manipulação de números decimais

## Tecnologias utilizadas

- C#
- Console Application


## Objetivo

Projeto desenvolvido para praticar lógica de programação e manipulação de arrays em C#.
